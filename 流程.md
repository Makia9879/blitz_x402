1. 用户在 UI 充值额度，获取 accessToken
2. 用户将 accessToken 设置到 claude code，开始使用 claude code
3. claude code 携带 accessToken 访问后台，调用大模型，后台统计每次调用消耗的 token 数
4. 后台调用大模型的接口检测到 token 余额不足，返回 httpStatus:402 响应
5. claude code 调用 MCPtool 向 failicator 支付 USDC，获取支付凭证
6. Claude code 调用 MCPtool 向后台返回 failicator 给的支付凭证，后台向 failicator 检查凭证有效性，有效则给用户添加 token 余额
7. 完成充值
