# 项目CC中转站

开发一个 CC 中转站，用于提供 CC 的中转服务。

# 功能

- 用户可以使用 MCP 工具为自己充值中转站使用 CC 的 token 额度
- 用户充值过额度之后，中转站分配对应额度，并开发使用 CC 的 token，并做额度使用统计
- MCP 工具体现 x402 支付协议
- 中转站对接模型实现提供大模型 CC 服务

# 系统组件

- MCP 工具：通信中转后台，实现 x402 支付协议
- 中转站 UI：额度展示、用数字钱包插件充值额度
- 中转站后台：调用智能合约、记录额度充值、统计额度使用、展示额度
- 智能合约：用于转账

# 技术风险

- x402 是什么？项目中怎么体现 x402？以何种形式展示 x402？
